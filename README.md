## Exercise 1

**Problem:**

Turn every number of a list into its cube.

**Example:**

 	Input: [1, 2, 3]
	Output: [1, 8, 27]
	
	Input: [4, 5, 6]
	Output: [64, 125, 216]
	
	Input: [0, -1, -2]
	Output: [0, -1, -8]

## Exercise 2

**Problem:**

Remove duplicates from list and return new list.

**Example**
	
	Input: [3, 2, 3, 1, 2, 3, 4, 2]
	Output: [3, 2, 1, 4]
	
	Input: [1, 2, 3, 4, 4, 5]
	Output: [1, 2, 3, 4, 5]
	
	Input: [5, 5, 5]
	Output: [5]

## Exercise 3

**Problem:**

Reverse the elements of a list.

**Example**

	Input: [1, 2, 3, 4]
	Output: [4, 3, 2, 1]
	
	Input: [5, 4, 3]
	Output: [3, 4, 5]
	
	Input: ['a', 'b', 'c']
	Output: ['c', 'b', 'a']
